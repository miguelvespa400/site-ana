# 💕 Site Para Ana - Minha Princesa

Um site elegante e romântico criado com amor para a pessoa mais incrível que conheço.

## ✨ Características

- 🎨 **Design Elegante e Moderno** - Interface visual impressionante com gradientes animados
- 🎠 **Carrossel de Fotos** - Galeria interativa com suas melhores fotos
- 🎭 **Fundo Animado** - Gradientes que mudam suavemente para um efeito hipnotizante
- ⭐ **Efeitos Especiais** - Estrelas e corações flutuantes que criam magia
- 💬 **Mensagens Carinhosas** - Palavras especiais e apelidos doces
- 📱 **Totalmente Responsivo** - Funciona perfeitamente em todos os dispositivos
- ✨ **Animações Suaves** - Transições elegantes e fluidas

## 🎯 Seções

1. **Cabeçalho** - Mensagem principal: "Para Você, Minha Princesa"
2. **Galeria** - Carrossel com as 4 melhores fotos
3. **Mensagens** - Cartões com mensagens especiais:
   - Minha Pituca 💕
   - Minha Rainha ✨
   - Meu Amor 🌹
   - Minha Gata ❤️
4. **Razões** - Por que você é incrível (6 razões especiais)
5. **Carta de Amor** - Mensagem final do coração

## 🚀 Como Usar

1. Abra o arquivo `index.html` no seu navegador
2. Clique nas setas para navegar no carrossel
3. Clique em qualquer lugar para criar corações flutuantes
4. Aproveite a experiência mágica!

## 📁 Estrutura do Projeto

```
site-ana/
├── index.html       # Arquivo principal
├── style.css        # Estilos e animações
├── script.js        # Funcionalidades interativas
└── img/             # Pasta com as fotos
```

## 💻 Tecnologias

- HTML5
- CSS3 (com animações avançadas)
- JavaScript Vanilla
